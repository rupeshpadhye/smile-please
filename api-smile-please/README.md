## Smile Please API 

## Description
This API fetches trending memes from 9gags , Imgur every hour and store it to firestore

## Prerequisite
- Need to create the project in firestore and add `firebase_service_account.json`
- Replace <IMGUR_CLIENT_API_KEY> in .env file  with imgur client key 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
